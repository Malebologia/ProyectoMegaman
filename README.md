MegaMan NES Intro — Proyecto Web Retro

Este proyecto recrea una interfaz retro estilo NES (Nintendo Entertainment System) inspirada en el mundo de MegaMan, con menú inicial, HUD, efectos CRT, música de fondo y navegación interactiva.

Diseñado para verse bien en PC, tablets y celulares.

Características del Proyecto

Estilo visual inspirado en la NES clásica.

Menú retro completamente funcional.

Navegación con teclado (⬆️ ⬇️ Enter).

Música de fondo con control.

Efecto CRT con scanlines.

Diseño responsive.

Separación en HTML + CSS.

Preparado para Git.

Tecnologías utilizadas

HTML5

CSS3

Google Fonts – Press Start 2P

Efectos CRT usando pseudo-elementos

VS Code como entorno de desarrollo

Git y GitHub (opcional)
